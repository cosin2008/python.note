    内嵌代码¶

    语法／示例 用途 效果
    `some code` 文本行内嵌代码
    `!#py3 some code` 文本行内嵌代码 代码以!#py3 开头可配合插件内嵌高亮显示

    行内嵌入代码显示效果

    代码块分页¶

    配合pymdownx.superfences插件可以实现代码块分页效果

    | TAB | 代码块 |

    需安装和启用插件。

    def hello(name): print('hello' % name) ```
